### Geral

\
Ao acessar a tela (via menu esquerdo ou atalho), é apresentado todos os cadastros já realizados, mostrando seus principais dados, o tipo - Cliente ou Fornecedor - e as ações disponíveis, baseando-se no cadastro de [permissões](/mapos/configuracoes/permissoes) do [usuário](/mapos/configuracoes/usuarios).

<Note>
  Caso não tenha nenhum cliente cadastrado, será informado inexistencia de cadastro.
</Note>

![Print da tela geral dos clientes](/images/clientes/grade-clientes.png)

### Adicionar cliente

Para iniciar o cadastro de um cliente, basta clicar no botão verde na tela geral do cliente.

Ao clicar, será redirecionado para a tela abaixo, onde todos os itens marcados com o **\*** (asterisco) é de preenchimento obrigatorio, ou seja, caso estejam em branco, não será possivel salvar o cadastro.

<Tip>
  Além dos dados obrigatórios pelo sistema, também indicamos cadastrar os números de celular e telefone para se usar a notificação via WhatsApp.
</Tip>

![Tela de cadastro de cliente](/images/clientes/adicionar-clientes.png)

<Tip>
  O sistema Map-Os conta com uma busca automatizada dos dados, desde que o cliente/fornecedor seja Pessoal Jurídica (CNPJ). - Basta digitar o CNPJ e clicar em Buscar (CNPJ) e os dados serão preenchidos para validação.
</Tip>

![Cadastro Cnpj Pn](/images/clientes/cadastro-cnpj.png)

Após adicionar e/ou validar os dados do cadastro, basta clicar em **SALVAR**. Ocorrendo tudo certo com a inserção dos dados, você será levado a tela inicial dos clientes com o aviso de sucesso.

![Salvo Sucesso Pn](/images/clientes/salvo-sucesso.png)

E a partir daí, já é possível utilizar este cadastro para realizar [vendas](/mapos/vendas) e [ordens de serviços](/mapos/ordens-de-servico).

### Visualizar cliente

Nossa primeira ação da grade, é a de **Ver mais detalhes**, ao clicar, somos levados para a tela abaixo, onde é exibido os dados gerais do cliente, tanto como suas Ordens de Serviço e Vendas realizadas pela empresa para ele.

![Dados gerais do cliente](/images/clientes/dados-geral.png)

Aba de Ordens de Serviço

Aba de Vendas

<Tip>
  Nas abas Ordens de Serviços e Vendas, há mais dois botões de ações, que pode nos levar a visualizar o movimento ou edita-lo.
</Tip>

### Área do Cliente

Nosso segundo botão de ação da tela inicial do cliente, nos leva para a Área do Cliente, essa tela estará disponível para clientes com E-mail e Senha cadastrados na hora da adição deles.

![Painel Cliente Pn](/images/clientes/painel-cliente.png)

### Editar cliente

O terceiro botão de ação nos leva para a tela de edição do cliente, tela similar a de cadastro, onde será carregado todos os dados já cadastrados dele.

![Editar Cliente Pn](/images/clientes/editar-cliente.png)

Após alterar o necessário, basta clicar em **Atualziar**.

### Excluir cliente

Ao clicar em excluir, aparecerá um alerta informando o risco dessa ação.

![Excluir Cliente Pn](/images/clientes/excluir-cliente.png)

<Warning>
  Ao decidir excluir, como informado na imagem, todos os movimentos do clientes será excluido, tenha cuidado ao usar esta função\!

  Valide sempre nas permissões, quem pode executar tal ação.
</Warning>
